Firedrake-Fluids
================

A collection of numerical models for fluid flow simulation using the [Firedrake](http://www.firedrakeproject.org) framework.

Dependencies
------------

Firedrake-Fluids depends on:
* [libspud](https://launchpad.net/spud)
